# 🏏 IPL Data Analysis Dashboard

## 🔗 Live App
https://ac4694-ipl-data-analysis-app-nht3pr.streamlit.app

---

## 📊 Overview
This project is an interactive IPL Data Analysis dashboard built using Python and Streamlit. It provides insights into team performance, toss impact, match trends, and venue statistics through dynamic visualizations.

---

## 🚀 Features
- 📅 Season-based filtering
- 🏏 Team-based filtering
- 📊 Interactive charts using Plotly
- 🥧 Win distribution (Pie chart)
- 📈 Matches per season analysis
- 🏟️ Venue analysis
- 📄 Raw dataset view

---

## 🛠️ Technologies Used
- Python
- Pandas
- Plotly
- Streamlit

---

## 🧠 Key Insights
- Mumbai Indians and Chennai Super Kings are the most successful teams
- Toss has around 50% influence on match outcomes
- Match distribution varies across seasons
- Some venues host more matches than others
